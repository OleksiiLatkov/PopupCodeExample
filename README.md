Code example for TellurionMobile interview. No Unity project, just few classes 

Simplified popups - without usual DI container and PopupManager, just to show something quick and light. 

MonoBehaviourWrapped - additional class to wrap MonoBehaviour with basic fields caching  
BasePopup class - base abstract class, with basic functionality
TwoButtonsDialog - class for popup with two configurable buttons (Yes/no, Ok/Cancel). Text for buttons and action handlers should be passed as Popup parameters.
